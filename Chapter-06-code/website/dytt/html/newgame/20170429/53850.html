经典单机游戏《狂扁小朋友》PC版

    发布时间：2017-04-29  

    中文名称: 狂扁小朋友
    游戏类型: 休闲益智
    游戏语言: 简体中文

    游戏简介:
    狂扁小朋友介绍: 一款画风独特、操控性佳的横向卷轴街头快打动作游戏，内容有点暴力又带点血腥。有一天暴力爸爸带着儿子经过和平镇的小公园，爸爸把儿子放下车，学习怎么跟附近的小朋友相处。 你要操纵紫色的暴力儿子，把看到的小朋友都打得半死不活
    键盘控制，方向键移动，A/S为轻拳重拳攻击，也可以跑步进行攻击，如果你熟悉了操作，那接下来还有各种技巧，可以施放各种连击哦.～
    普通控制：A：轻拳，S:重拳。方向键：走路，跑步：连按两下左箭头（←）或是连按两下右箭头（→）。
    连招：狂抽技：既然是狂抽技，当然就是抽到那些小朋友毫无还手之力的技能，施放的方法也很简单，你只要连按三下A，然后立刻按两下B，如此重复按，可以一直打到小朋友挂掉.
    《狂扁小朋友》必杀技:
    1.风车拳：连按←←（连按→→）＋S
    2.抱拳：连按←←（连按→→）＋A
    3.风车拳：连按←←（连按→→）+SSSSS……
    4.小刀：先按A再立即按S
    5.双连刀：A+S+A
    6.抓住：走路到小朋友跟前即可
    7.上勾拳：A+S（一起按）
    8.摔：抱住小朋友+←（→）+A
    9.投掷：抱住小朋友+←（→）+S
    10.双拳重击：A+S+A+S+S
    11.二连击：A+A(S+S)
    《狂扁小朋友》少见绝招:
    拳脚大连环：A+A+A+A（头顶）+S（脚蹬）+S+S（双手捶）
    踩：小朋友倒地时到他身上+A（S）
    双重爆气：A+A+A+……+S+S+S(直到爆气）
    1)沙滩上的小朋友
    一开始，爸爸开车带着你（顺便撞飞了一个小朋友），下车后，见一小孩，游戏开始。 提示：踩掉沙堡，那个小朋友会哭。 注：小朋友无攻击力。
    2)游乐场
    到了幼儿园，见一幼儿园老师也在"狂扁小朋友"，还打死了一个。 这家伙便是第一个BOSS，攻击力：10 ，血厚，建议用狂抽技，打死为止。
    提示：有个小孩骑的木马是会随机变化的。
    3）公园秋千
    提示:可以把小孩打、撞飞。注意：戴头盔的小孩有攻击力！
    4）足球场 　　
    提示：可以把球踢进球门，还会加分（分数在背景上）。当你持足球时，小孩会铲你的球。
    5）公路
    这里有五个小朋友准备过马路（当然，他们永远也不会过去的）， 他们一动不动站在那里反而更好打。
    提示：从公路上开来的汽车具有较强的攻击性，当然，也伤小朋友。
    6）草坪
    又有好多小朋友来了！其中有一个黑色西装的小子会拿一把玩具枪射你（子弹是马桶把子？），被打中会伤血，但射出去的子弹可以用拳头打掉。 提示：停在右上方的那个装油桶的推车也可以当炸药使哦！
    7）篮球场 　　
    虐待人家中学生！提示：持篮球时，按↑+A（S）可以投篮（只要你在篮框左边，不管怎么投都会中哦！） 　
    8）弄堂 　
    有两个小孩开着车来了！可以用篮球打爆或用上面的垃圾桶，还可以用甩拳或风车拳！
    提示：把垃圾桶夹在角落里，狂扁它会增加爆气率。
    9）小花园
    这一关，场地比较大，人也比较多，多爽几下吧，用用刀子和下勾拳，增加增加自己的狂暴率！
    10）小区边缘
    注意：那两个忍者小孩很牛B！血也比普通小朋友的厚！建议用→+→+S+S+S+，有一个忍者小孩开着车哦！
    11)最终BOSS黑社会领地 　
    决战时刻到！超级牛B的BOSS出现血厚的要命，建议用A+S之类的连招打！如果血不是所剩无几，见到BOSS前的在地上的那个血包千万不要吃！注意：他被放倒时别靠近他，还要小心他的大绝招——影连环倒立分身术！（被踢中非常伤！）上面的两个油桶也是一个支援哦！
    开始会看见Boss在玩飞腿，并给手枪上了子弹。不要管他，先去煤气罐旁边，他会跟去，再回到他练飞腿的地方对着他投掷篮球，这时，赶紧去捡会足球。注意：千万不要和他一条线，否则，他会连续射枪，让你起不来，那可就惨了！要知道，他的枪可是比得上威力一百五的破坏死光的！这次开始要多用小刀、下勾拳，直到狂暴为止，再狂用甩拳，在他刚刚倒地，还没来得及起来时，再给他来一个甩拳，让他也常常被打得起不来的滋味！但是要掌握好时机，一旦有一次没有打中他，那他就会起来再"报复"你——用手枪连射或用超级百万飞腿。
