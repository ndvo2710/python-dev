经典单机游戏《罗刹》PC版

    发布时间：2017-09-01  

    中文名称: 罗刹
    游戏类型: 即时战略
    游戏语言: 简体中文

    游戏简介:
    中文名称: 罗刹
    英文名称: Rasetsu
    游戏类型: RTS 即时战略游戏
    发行时间: 2001年12月07日
    制作发行: 金马科技(有限)公司出品地区: 大陆
    语言: 简体中文

    距今3000年后的世界，当时宇宙分为两大势力：地球为中心的"地球联邦"和反对派"迪克行星联盟"。在长久的战争中，诞生了不少以杀戮为职业的人，人们将这些人称之为"佣兵工会"。玩家扮演的就是"佣兵工会"中的一员，游戏的每一关都要从两大势力处接受各种各样的任务，完成战斗使命。玩家在每战之前可以利用有限的资金雇佣战士，调配生化装甲，并且对同伴的能力进行训练，在实战中便可对雇佣兵们进行有效的指挥。
    在战斗中，以肉身迎击对手的钢铁洪流是不现实的，佣兵们必须借助生化装甲的帮助。生化装甲也分成3个等级。一开始佣兵只能拥有最初级的装甲，但随着研究资金的投入，能研发出其他的高等级生化装甲。高级的生化装甲还可能拥有特殊的能力，譬如飞行。飞行兵种的出现到是让最终的战斗格局显得立体感十足。令人高兴的是，由于这个非常基本的设定，只要熬过最初的一段时间，每个玩家都能组合出适合自己战术的部队。

    每名佣兵都有6项战斗属性：力量、灵敏度、命中率、防御力、战斗力和行动力。要提升这6样属性可以对佣兵进行药物强化。不过要获得强化的资格还是要由战斗经验来决定。玩家完全能把这个系统当作升级系统来看待。在经验获得到一定程度后，还能在佣兵学校里获得"执照"。这些执照能提升佣兵特定的战斗能力，十分有用，建议每个玩家冲关时给每个佣兵都弄一个。

    在战斗进行的过程中间有商店这个建筑感念存在。不过就同前面所说的，这里是不卖主要的射击武器的。玩家在这里只能买到些辅助攻击道具。像手榴弹或者陷阱装置等。这些道具虽然不会大幅度增强雇佣兵团队的战斗能力，但在特殊的环境下，能发挥出强大的功效。
    在整个游戏中间，玩家要考虑的只是如何在战场中间用最快的时间消灭敌人，如何控制自己少量的部队来避开对手的大部队而直达目标，如何在最短的时间内完成最多的任务来增加原始资金的积累。总之一句话，玩家要考虑的只是屏幕中间的士兵。
    于之对应的是，在每一关的开始之前的种种设定让游戏隐约有了策略制的特征，在一场战斗结束以后强化自己的装甲，升级自己部队的战斗力……是不是有一点想到了机器人大战？这样的设定导致的直接结果就是原本即使战略中需要玩家在战斗中开发的一些东西现在在战斗前都已经准备好了。而在战斗中间玩家所要做的事情就是最最基础的消灭敌人，摧毁目标。
