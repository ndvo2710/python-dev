经典单机游戏《梦幻海洋公园》PC版

    发布时间：2017-09-01  

    中文名称: 梦幻海洋公园
    游戏类型: 休闲益智
    游戏语言: 简体中文

    游戏简介:
    中文名称：梦幻海洋公园
    中文别称：海洋公园帝国
    英文名称：Marine Park Empire
    发行时间：2005年
    游戏制作：Enlight Software / BTpig
    游戏发行：光谱资讯 / BTpig
    游戏类型：SIM模拟经营
    游戏语言：繁体中文
    ：
    拥有大型水族馆里老少咸宜的海狮秀、海豚明星与杀人鲸表演永远都能够吸引游客的目光。"梦幻海洋公园"除了一般动物园类型游戏常见的生物外，游戏增加了15种各具特色的水中生物，再加上超自由的游戏方式，让玩家有机会能够体验不同于以往模拟经营陆上动物园新体验。
    海陆通吃的超大型动物园
    虽然名为"梦幻海洋公园"，但玩家一开始的教学关卡却是在一座小型的"陆上"动物园。玩家要先从照料陆上的动物开始着手，随着关卡的进行，很快玩家就能够经营水族馆，甚至是混合型态的大型动物园。游戏中要注意的项目非常多，从地形的设定，动物的安置，园内的花草树木、贩卖
    ，甚至游客参观的路线，都由玩家来决定，必须好好的注意各项细节。要是在淡水鱼缸里放了海水，下场可不是鱼儿翻肚子这么简单喔！
    全方位的经营管理
    除了栩栩如生的3D场景环境外，游戏在经营管理的模式上也下了很大的功夫。无论是动物的喂食、照顾、生态环境的配合上，都要仔细考虑，而这些经营的细节都能够在游戏中的20多个任务关卡中慢慢学到。虽然有许多擅长这类型经营游戏的玩家都比较喜欢玩自由模式来创造自己的海洋公园，但"梦幻海洋公园"中却有许多动物与设备，是在通过任务关卡后才能取得的。所以玩家们在刚开始进行游戏时，先别急着进行自由模式，在任务模式累积一下实力并取得隐藏的动物与设备，游戏进行的乐趣会更为提高喔～
    动物开心→游客满意→提高获利
    在任务关卡中，除了主要目标外，还有些次要的任务，虽然不影响过关的条件，但是完成后却会有些特别的奖励。除了金钱之外，也有可能是特殊的动物或设备，而这些物品都是可以带入下一个关卡使用的。除此之外，玩家可以训练聪明的动物成为原内的"动物明星"，藉以招揽更多的游客。总体而言，只要玩家妥善地照顾园内的动物，让游客们看到他们健康活泼的一面，游客们也就不会吝啬让身为园长的你榨干他们的荷包～

    亲切的操作接口
    游戏公司在前作"ZOO EMPIRE"上市后，接受了许多玩家对于游戏改善上的意见，并忠实的反映在"梦幻海洋公园"中，操作接口变的更加人性化。此外，动物数据系统也依然健在，并且也有所改良。玩家在购买或查询动物的同时，可以看到在现实世界中的生态信息等相关数据，不仅可以作为兴建居住环境的参考，更是充满了教育的意味～说是"寓教于乐"还真一点也不为过。
