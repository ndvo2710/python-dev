经典单机游戏《罗马执政官》PC版

    发布时间：2017-09-01  

    中文名称: 罗马执政官
    游戏类型: 即时战略
    游戏语言: 繁体中文

    游戏简介:
    中文名称: 罗马执政官
    英文名称: Praetorians
    别名: 君临天下
    游戏类型: RTS 即时战略游戏
    发行时间: 2003年
    制作发行: Pyro Studios
    Eidos Interactive地区: 美国
    语言: 繁体中文

    《罗马执政官》是《盟军敢死队》原创小组Pyro Studios历时3年精心制作的一款即时策略游戏。与一般的即时策略游戏略有不同，《罗马执政官》并不强调建设和积攒资源，而是更多的偏向对单位的控制和战斗策略。这样在游戏整体上感觉战斗更直接和激烈。
    游戏的故事是以野心最大的恺撒、最富有的克拉苏和声望最高的庞贝结为"三巨头"同盟开始，以恺撒在生前的最后一战为结尾，整体气势极为宏大，时代背景也刻画得也很深厚。游戏中罗马共和国晚期的不断扩张和频繁的内战都被表现出来，游戏里主要能控制的都是归属于恺撒的军团，玩家将随着他们多次征服高卢全境、远征不列颠、转战小亚细亚、最终登陆埃及。虽然游戏全程大都是对外族的征战，但是三位执政官间的明争暗斗乃至兵戎相见都时时能体会到。当完成整个剧情游戏时，玩家会体会到与伟大统率恺撒一样的成就感。
    像现在所有即时战略游戏一样，《罗马执政官》除了单人剧情模式，也包括多人模式。多人模式中可以选择罗马人、野蛮人和埃及人。这就是那个时代活跃在欧洲和北非大陆上的主要军事力量。罗马人的部队特点是纪律严明装备精良，战斗力很强不过也意味着训练时间长。野蛮人主要指的是那个时代的高卢人和日尔曼人，他们单兵作战能力都极强，适合利用地形打伏击战，大规模作战则稍弱。埃及人的作战能力不是很强，不过他们部队的建造时间会很短，并且拥有帕提亚骑兵这个强力兵种。游戏中设计的三个阵营战斗力十分平衡，不同兵种的相克也十分明显，在战斗中各有所长，于是指挥官的战术运用就成了多人模式中取胜的关键。
    《盟军》的创作者们将把玩家带入了罗马共和国最后几年的战争时代。你将指挥着恺撒的罗马军团与高卢人、埃及人和其他罗马人对抗。 《罗马执政官》游戏有20多个任务。游戏开头是朱利叶斯·恺撒征服了高卢人，接着经历内战和埃及战役。在整个游戏，玩家将扮演一名罗马将军的角色，偶尔也有机会在为了至高权利的战斗中指挥高卢和埃及的军队。游戏的任务根据实际的战争情况而定。
    《罗马执政官》游戏删除了典型的建造房屋和采集资源的模式，把注意力集中在战斗。在每个游戏开始时，玩家将有一定数量的军队。这正是罗马战斗的风格，你从不需要担心他们会被杀死。同时在执行任务前，游戏会为玩家提供完整的地图，让玩家对地理形势了如指掌。如果玩家只能在地图上看到敌军的话，玩家就必须四周侦察，直到你可以比较清晰可见敌军的具体位置。
     一组组的军队移动起来就像是单一的实体。玩家不必再把军队套索在一起，然后编队移动他们。玩家只需选好一个单独军团就可以整体移动了。如果玩家喜欢的话，也可以把军队分成小组。军队分成的组数将会以罗马数字在屏幕的上方显示出来。
    轻步兵团是军队中最弱的军队，但是他们可以好好利用于修理和建造方面，也就是可以建造城堡，熔练武器。重甲兵装备了长矛，可以阻挡敌人进攻。重步兵和骑兵组成的突击对和射兵队，给游戏增添了不少的乐趣。 每个军队都有特殊的能力，但是只能使用一次，用完后必须再补充。射兵队可以切换到联锁姿态，防止他们移动，这就大大地扩大了他们的范围和提高他们的准确性。重步兵有短矛，可以向远距离的敌人飞射过去。高卢的步兵也使用同样的战术，效果如何那就不清楚了。军队的其他特殊的能力持久力有限，所以玩家要执行其他能力，或者快速横越地图。持久力会随时间而慢慢补充，但玩家最好还是合理保存，一旦需要用的时候就可以用到。 游戏里还有一些拥有他们特有的能力的军队。例如，百夫长可以出现在战斗中，提升军队的士气。也可以在玩家的命令下进入城镇，训练平民成为士兵。而探测队有只老鹰，可以利用它侦察地形。老鹰可以向军队提供情报，告知森林的另一边有敌人的投石器。但是罗马人也很聪明，能分辨出是敌人的老鹰，所以毫不留情杀死它。
    除了这些特殊的能力外，不同军团也有独特的优势。罗马军团拥有骄人的组织性和纪律性。他们会合理利用道路，形成更大的编队。他们的屏障编队可以带来更大破坏，但又防止了士兵的移动和敌方射兵的入侵。而高卢军团就没那么老练了。他们比罗马军团移动的快，破坏也更大，但是他们没有编队，一般都是小组移动。
    在战术方面来说，《罗马执政官》游戏主要是看玩家战前的组织，而不是战斗中的实际操作。为了符合历史的准确性，军团一旦作战，就不可能随便调遣改变。尽管如此，个别的混战短而激烈，因此玩家有很多的时间重组和再编制。 为符合3D华丽新潮流，《罗马执政官》游戏的图形画面形象生动。3D的地形，地面的结构精细，物体栩栩如生，海水轻轻拍打着海岸，茂密的森林把你带进游戏世界。 防御也是游戏的一大特色，所以玩家要么攻要么守。阵营和城镇非常适合一些紧张的围攻和街头交火。有进入到城镇，玩家就要利用一些围攻武器，可以召集军队围攻城镇，爬墙而入，也可以利用发石器远距离的发射，这一招还能杀死不少的敌人呢。 游戏的摄像镜头是固定的，有点让人失望。当不管怎样，玩家还可以少许倾斜查看周围的环境。
